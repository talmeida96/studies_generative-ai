# PÓS GRADUAÇÃO IA GENERATIVA - UFG

### CRIAR AMBIENTE VIRTUAL COM O MINIFORGE:

- Criar ambiente virutal:
```sh
conda env create -n generative_ai -f environment.yml
```

- Ativar ambiente:
```sh
conda activate generative_ai
```
